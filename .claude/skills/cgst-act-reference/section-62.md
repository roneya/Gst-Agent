# Section 62 — Assessment of Non-Filers of Returns

## What Is Section 62?

Section 62 empowers the proper officer to assess the tax liability of a registered person who **fails to file their return within 30 days of a GSTR-3A notice** (issued under §46).

This is called a **Best Judgement Assessment (BJ Assessment)** — the officer uses all available information to estimate the tax liability in the absence of the taxpayer's actual return.

---

## The Section 62 Trigger Chain

```
GSTR-3B / GSTR-1 not filed
        ↓
GSTR-3A notice issued under §46 (15-day window to file)
        ↓
Taxpayer does not file within 15 days of GSTR-3A
        ↓
Proper officer passes ASMT-13 (BJ Assessment) under §62
        ↓
Tax demand based on officer's estimate
```

---

## Key Features of Section 62

| Feature | Detail |
|---|---|
| Trigger | Non-filing after GSTR-3A notice |
| Assessment form | **ASMT-13** |
| Basis of assessment | Officer's best judgement — e-way bills, TDS/TCS data, GSTR-1 of suppliers, banking data |
| Demand | Typically much higher than actual liability |
| Remedy | File actual return within 30 days of ASMT-13 → automatic withdrawal |
| Post-30-day remedy | Appeal under §107 or writ petition |

---

## The Critical Remedy — §62(2)

> **"Where the registered person furnishes a valid return within thirty days of the service of the assessment order under sub-section (1), the said assessment order shall be deemed to have been withdrawn."**

This is the most important provision — **filing the actual return within 30 days of ASMT-13 automatically withdraws the BJ Assessment.** No appeal, no hearing needed.

---

## Immediate Steps on Receiving ASMT-13

1. **Note the date of service** — 30-day countdown to file actual return starts immediately
2. **Urgency flag** — If < 10 days remaining → file return TODAY
3. **Compute actual tax liability** — Compare with ASMT-13 demand (usually overestimated)
4. **File all pending returns** — Must include all periods covered in ASMT-13
5. **Pay actual tax + interest + late fee** — Required before filing
6. **ASMT-13 automatically withdrawn** under §62(2)

---

## What If 30 Days Has Passed?

If the taxpayer fails to file within 30 days:

- ASMT-13 becomes **final and enforceable**
- Recovery proceedings under §79 can commence
- Options:
  1. **Appeal to Appellate Authority (§107)** — file APL-01 with 10% pre-deposit
  2. **Writ petition** — if the BJ Assessment is arbitrary / based on wrong data
  3. **Negotiate with officer** — file actual return + request withdrawal (discretionary, not a right after 30 days)

---

## How Officer Estimates Tax in BJ Assessment

Common data sources used:
- GSTR-1 filed by the taxpayer's customers (showing taxpayer's sales)
- E-way bills generated by / for the taxpayer
- TDS/TCS credits in taxpayer's credit ledger
- Bank statements obtained under information gathering powers
- Comparison with previous periods' returns
- Industry-specific turnover norms

> **Result:** The ASMT-13 demand is almost always **significantly higher** than the actual liability — filing the actual return quickly is always in the taxpayer's interest.

---

## Filing the Return After ASMT-13

### Step 1 — Calculate actual liability
- Compute output tax for all pending months
- Compute eligible ITC
- Compute net tax payable in cash
- Add interest @ 18% from original due date
- Add late fee

### Step 2 — File pending returns on portal
```
Services → Returns → Returns Dashboard
→ Select period → File GSTR-1 first → then GSTR-3B
→ Pay tax + interest + late fee
→ Submit
```

### Step 3 — Inform the officer
After filing, send a communication to the officer with:
- ARN of the filed returns
- Challan reference for tax paid
- Formal intimation that §62(2) remedy has been exercised and ASMT-13 stands withdrawn

## ASMT-13 vs ASMT-10 — Key Difference

| Aspect | ASMT-10 (§61 Scrutiny) | ASMT-13 (§62 BJ Assessment) |
|---|---|---|
| Trigger | Return mismatch | Non-filing |
| Reply form | ASMT-11 | File actual return (§62(2)) |
| Tax demand | Not yet (pre-demand) | Yes — demand confirmed |
| Withdrawal | ASMT-12 on acceptance | Automatic on filing return within 30 days |
| Urgency | 30 days to reply | 30 days to file return |

---

## Key Legal Points

1. **ASMT-13 is not a final order** if filed within 30 days — §62(2) is a complete remedy.
2. **Interest still runs** from the original due date even after ASMT-13 withdrawal.
3. **Late fee still payable** — filing the return does not waive the late fee.
4. **BJ Assessment must be reasonable** — officer cannot use arbitrary figures; available data must be used as the basis.
5. **GSTR-3A is mandatory pre-condition** — if GSTR-3A was never issued, ASMT-13 is procedurally invalid.

---

## Key Case Law

- *Tvl. Kavin HP Gas Gramin Vitrak v. CTO* (Mad HC) — §62(2) remedy is absolute; filing return within 30 days withdraws assessment automatically
- *SKH Sheet Metals Components v. UOI* — BJ Assessment must be based on available material; cannot be purely arbitrary
- Multiple HCs: GSTR-3A is a mandatory condition precedent to §62 assessment
