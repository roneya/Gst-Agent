## Portal Filing Path

```
Services → Registration → Application for Filing Clarifications
→ Select REG-03 reference number
→ File REG-04
→ Upload all documents as attachments
→ Submit
```

**Deadline:** Within **7 days** of REG-03 notice date.

---

## Common Documents Required

| Query Type | Documents |
|---|---|
| Principal place of business | Electricity bill / property tax receipt + rent agreement / ownership deed + NOC |
| Identity of proprietor/director | Aadhar + PAN + photograph |
| Business activity | Purchase/sale invoices + website / letterhead + business registration |
| Bank account | Cancelled cheque or first page of passbook |
| Additional place of business | Separate address proof for each place |
| Partnership / company | Partnership deed / MOA + AOA + board resolution |
