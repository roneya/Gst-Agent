# ASMT-11 Reply Template

## Filing Path
Portal: Services → User Services → My Applications → Reply to Notice → Select ASMT-10 → File ASMT-11

---

---

## Checklist Before Filing

- [ ] All discrepancies from ASMT-10 addressed (none skipped)
- [ ] DRC-03 ARN quoted for all accepted discrepancies
- [ ] Annexures numbered and listed
- [ ] Reply filed on portal (ASMT-11) AND sent physically/email to officer
- [ ] Filed within 30 days of ASMT-10 service date
- [ ] Extension request filed if deadline is close
