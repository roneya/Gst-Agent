# REG-03 & REG-17 — Registration Notices

---

## PART 1 — REG-03: Query / Clarification Notice on Registration

### What Is REG-03?

REG-03 is a **notice seeking clarification or additional documents** during or after the GST registration process. It is issued by the proper officer when:
- The registration application (REG-01) needs additional information / documents
- The officer is not satisfied with the documents submitted
- Post-registration verification reveals discrepancies

- Governed by: **Section 25** of CGST Act, 2017
- Rule: **Rule 9(2)** of CGST Rules, 2017
- Reply form: **REG-04**

### Immediate Checklist on Receipt of REG-03

1. Note the **7-day response window** — REG-04 must be filed within 7 days
2. Read the specific queries/documents requested
3. Gather all supporting documents
4. Urgency flag: if < 3 days → respond immediately

### Common REG-03 Queries

| Query | Documents to Submit |
|---|---|
| Principal place of business proof | Electricity bill, rent agreement, NOC from owner |
| Identity proof of proprietor/director | Aadhar, PAN, passport |
| Business activity proof | Purchase invoices, website, letterhead, business registration |
| Turnover threshold clarification | Bank statements, financial summary |
| Additional place of business | Separate address proof for each location |

### How to File REG-04 (Reply to REG-03)

1. Prepare all documents (self-attested copies)
2. File REG-04 on GST portal within 7 days
3. Upload supporting documents as attachments

**Portal Path:**
```
Services → Registration → Application for Filing Clarifications
→ Select REG-03 reference → File REG-04 → Upload docs → Submit
```

### Consequences of Not Responding

- If no reply within 7 days → officer **rejects the registration application** under REG-05
- Fresh application (REG-01) must be filed after rejection

## PART 2 — REG-17: Show Cause Notice for Cancellation of Registration

### What Is REG-17?

REG-17 is a **Show Cause Notice for cancellation of GST registration** issued by the proper officer. It is the most serious registration notice — if not replied to properly, the GSTIN can be cancelled.

- Governed by: **Section 29(2)** of CGST Act, 2017
- Rule: **Rule 22** of CGST Rules, 2017
- Reply form: **REG-18** (reply to REG-17)

### Common Grounds for REG-17

| Ground | Details |
|---|---|
| Non-filing of returns | 6+ consecutive months GSTR-3B not filed (monthly) / 2+ quarters (quarterly) |
| Registration obtained by fraud | Fake documents, benami registration |
| Business discontinued | Business no longer being carried on |
| Change in constitution without intimation | Partnership dissolved, proprietor deceased |
| Not conducting any business | Dormant registration |
| Voluntary cancellation pending | Taxpayer's own REG-16 application under process |

### Immediate Checklist on Receipt of REG-17

1. Note the **7-day response window** — REG-18 must be filed within 7 days
2. Identify the specific ground cited in REG-17
3. Check if pending returns need to be filed first (non-filing ground)
4. Check if DRC-03 payment is needed before responding
5. Urgency flag: GSTIN suspension may already be active — verify portal status

### Response Strategy

**If ground is non-filing of returns:**
1. File ALL pending returns immediately (GSTR-1, GSTR-3B with late fee + interest)
2. File REG-18 attaching proof of all filed returns (ARNs)
3. Request that SCN be dropped since default is now cured

**If ground is business discontinued / change in constitution:**
- If correct → consider voluntary cancellation via REG-16 instead of contesting
- If incorrect → provide proof of business continuity (recent invoices, bank statements, utility bills)

**If ground is fraud / benami:**
- This is a serious allegation — engage a CA / advocate immediately
- File REG-18 with strong documentary rebuttal
- Typically requires personal hearing

### How to File REG-18 (Reply to REG-17)

**Portal Path:**
```
Services → Registration → Application for Filing Clarifications
→ Select REG-17 reference → File REG-18 → Upload docs → Submit
```

### What Happens After REG-18?

```
REG-17 (SCN for cancellation)
    ↓
REG-18 filed by taxpayer
    ↓ Officer satisfied
REG-05 — Application rejected (no cancellation)
    OR
    ↓ Officer not satisfied
REG-19 — Cancellation Order
```

### If REG-19 (Cancellation Order) Is Issued

- The GSTIN is cancelled from the effective date mentioned in REG-19
- **Appeal:** File an appeal within 3 months under Section 107 (GST appeal to Appellate Authority)
- **Revocation:** If cancellation was for non-filing, taxpayer can apply for **revocation (REG-21)** within 90 days of cancellation order, after filing all pending returns and paying dues

### Revocation of Cancellation — REG-21

Conditions for revocation:
1. File all pending returns
2. Pay all outstanding tax + interest + late fee + penalty
3. Apply REG-21 within 90 days of REG-19 cancellation order
4. Officer issues REG-22 (revocation order) if satisfied

**Portal Path:**
```
Services → Registration → Application for Revocation of Cancelled Registration
→ File REG-21 → Attach proof of return filing → Submit
```
