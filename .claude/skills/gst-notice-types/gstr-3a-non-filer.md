# GSTR-3A — Notice to Non-Filer / Defaulter

## What Is a GSTR-3A?

GSTR-3A is a **system-generated notice issued to a registered taxpayer who has failed to file their GST returns** within the prescribed due date.

- Governed by: **Section 46** of CGST Act, 2017
- Rule: **Rule 68** of CGST Rules, 2017
- Issued by: The proper officer (often auto-generated by the GST system)
- Applies to: Non-filing of GSTR-1, GSTR-3B, GSTR-4, GSTR-5, GSTR-6, GSTR-7, GSTR-8, GSTR-9, GSTR-10

---

## GSTR-3A Notice — Key Facts

| Item | Detail |
|---|---|
| Notice type | Non-filing notice |
| Authority | Proper officer under §46 |
| Time given to comply | **15 days** from date of notice |
| Consequence of non-compliance | Best Judgement Assessment under §62 |
| Form for BJ Assessment | ASMT-13 |

---

## Immediate Checklist on Receipt

1. **Identify which return is pending** — GSTR-1? GSTR-3B? Which period(s)?
2. **Check notice date** — 15-day countdown starts from service date
3. **Urgency flag** — If < 5 days remaining → file the return immediately
4. **Check GSTIN status** — Is the registration active or suspended?
5. **Compute late fee** — Late fee is mandatory before filing (load `gst-thresholds`)
6. **Compute interest** — If GSTR-3B, interest on unpaid tax @ 18% p.a. under §50

## Late Fee Structure (as of 2024)

| Return | Late fee (CGST per day) | Maximum cap |
|---|---|---|
| GSTR-3B (NIL return) | ₹ 10 | ₹ 500 (CGST) |
| GSTR-3B (taxable return) | ₹ 25 | ₹ 10,000 (CGST) |
| GSTR-1 (NIL return) | ₹ 10 | ₹ 500 (CGST) |
| GSTR-1 (taxable return) | ₹ 25 | ₹ 10,000 (CGST) |
| GSTR-9 Annual Return | 0.25% of turnover per day | Capped |

*(Equal SGST late fee applies in addition — so multiply by 2 for total)*

## Response Options

### Option A — File the Pending Return (Recommended)

1. Calculate total late fee + interest payable
2. File the pending return on the GST portal with payment
3. After filing, the GSTR-3A notice is automatically resolved — no separate reply needed
4. Keep the ARN of the filed return as proof

**Portal Path:**
```
Services → Returns → Returns Dashboard
→ Select Financial Year + Period → File GSTR-3B / GSTR-1
→ Pay late fee + interest via cash ledger
```

### Option B — File the Return with Explanation Letter

If the non-filing was due to a genuine reason (server issues, bereavement, natural calamity), file the return AND send a covering letter to the jurisdictional officer requesting waiver of late fee under §128.

### Option C — Apply for Late Fee Waiver (If Amnesty Notified)

Check `gst-thresholds` SKILL.md for current amnesty notifications. If an amnesty scheme is active, file the return under the scheme to avail reduced / nil late fee.

## Consequences of NOT Responding

If the taxpayer fails to file within 15 days of GSTR-3A:

```
GSTR-3A (15-day notice)
    ↓ No response
Best Judgement Assessment — ASMT-13
(Officer assesses tax based on available information)
    ↓
Tax demand raised in ASMT-13
    ↓ Within 30 days of ASMT-13
Taxpayer can file actual return + pay tax
ASMT-13 automatically withdrawn
    ↓ After 30 days
ASMT-13 becomes final — appeal required to contest
```

---

## Best Judgement Assessment (§62) — ASMT-13

- Officer assesses tax liability using data available (e-way bills, GSTR-1 of suppliers, TDS/TCS data, etc.)
- The assessment can be **significantly higher than actual liability**
- **Remedy:** File the actual pending return within 30 days of ASMT-13 → officer shall withdraw ASMT-13 under §62(2)

---

## Multiple Periods Pending?

If GSTR-3A covers multiple periods:
1. File in chronological order — oldest period first
2. Each period's return must be filed separately
3. Some periods may require filing GSTR-1 before GSTR-3B can be filed

---

## Registration Suspension Risk

If a taxpayer fails to file 6 or more consecutive months' GSTR-3B (or 2 consecutive quarters for quarterly filers), the GSTIN may be **suspended under Rule 21A**. Check GSTIN status on the portal before responding.

---

## Relevant Sections

- **Section 46** — Notice to return defaulters (GSTR-3A)
- **Section 62** — Best Judgement Assessment for non-filers (ASMT-13)
- **Section 47** — Late fee for delayed return filing
- **Section 128** — Power of government to waive late fee
